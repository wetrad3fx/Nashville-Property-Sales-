# Nashville Housing Market Analysis

## Project Overview
End-to-end data analysis project using SQL and Power BI on 56,636 
Nashville property sales from 2013-2016.

## Tools Used
- PostgreSQL (data cleaning & analysis)
- VS Code + SQLTools extension
- Power BI Desktop (dashboard)

## Key Findings
- Average home price grew 24% from $242K (2013) to $300K (2016)
- Brentwood is the most expensive suburb at $316K average
- Single Family homes dominate at 60% of all sales
- 12,180 rows flagged as bundled multi-parcel sales

## Files
- Nashville_cleaning.sql — data cleaning queries
- Analyze.sql — analysis queries
- nashville_dashboard.pbix — Power BI dashboard
- nashville_cleaned.csv — cleaned dataset
